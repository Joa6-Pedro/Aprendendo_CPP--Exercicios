# Aprendendo_CPP--Exercicios
Exercícios de Pratica da linguagem C++ - Data de inicio 11/02/26

Os exercícios têm como objetivo praticar lógica de programação, controle de fluxo, entrada e saída de dados e estrutura básica da linguagem C++.

1 - Calculadora: Deve Requisitar ao usuário uma operação matemática basica a ser realizada por meio de valores numéricos. Após a entrada do valor deve requisitar outros dois valores a serem processados e retornados por meio do console. O programa deve ser finalizado quando na "tela" incial o usuário digitar o valor '0'.

2 - Jogo de advinha: O programa deve gerar um número aleatório dentro de um intervalo definido e solicitar que o usuário tente adivinhar qual é esse número. A cada tentativa, o sistema deve informar se o valor digitado é maior ou menor que o número secreto. O jogo deve continuar até que o usuário acerte o número, exibindo ao final a quantidade total de tentativas realizadas.

3 - Sistema de Login Simples: O programa deve simular um sistema básico de autenticação contendo um login e senha previamente definidos no código. O usuário terá um número limitado de tentativas para inserir corretamente as credenciais. Caso os dados estejam corretos, o acesso será liberado; caso contrário, após exceder o número máximo de tentativas, o sistema deve bloquear o acesso e encerrar a execução.

4 - Contador de Vogais em Frase:
O programa deve solicitar ao usuário a digitação de uma frase completa e realizar a análise de seus caracteres. O sistema deve contar o número total de letras (desconsiderando espaços e caracteres não alfabéticos), além de identificar e contabilizar a quantidade total de vogais presentes na frase, bem como a quantidade individual de cada vogal (A, E, I, O e U), independentemente de estarem em maiúsculas ou minúsculas.

5 - Análise de Notas com Vetor:
O programa deve solicitar ao usuário a entrada de um conjunto fixo de valores numéricos armazenados em um vetor. Após a entrada dos dados, o sistema deve calcular e exibir a média dos valores, bem como identificar o maior e o menor valor presentes no vetor. O exercício tem como objetivo introduzir o uso de vetores e percorrê-los utilizando laços.

6 - Operações com Vetor Utilizando Funções:
O programa deve trabalhar com um vetor de valores numéricos e utilizar funções para realizar operações sobre esses dados. O sistema deve permitir a inserção dos valores no vetor, calcular a soma dos elementos e identificar o maior valor armazenado. Cada operação deve ser implementada em funções separadas, reforçando conceitos de modularização e passagem de arrays como parâmetros.

7 - Sistema de Gerenciamento de Valores:
O programa deve implementar um sistema interativo baseado em menu para gerenciar um conjunto de valores numéricos armazenados em um vetor de tamanho fixo. O usuário deve poder inserir valores, exibir todos os valores armazenados, calcular a soma, identificar o maior e o menor valor, além de encerrar o programa. Todas as funcionalidades devem ser implementadas por meio de funções, utilizando controle de fluxo e laços de repetição.
